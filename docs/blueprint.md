# **App Name**: LocalAdmin

## Core Features:

- Admin Authentication: Basic login form with username 'admin' and password 'admin' for local access (no database).
- Admin Dashboard: A dashboard to manage users, forms, and data entries.
- User Management Module: A user management module to create and manage user profiles locally.
- Form Module: Form creation module to create custom data entry forms with various field types.  The types will need to be supported by Nextjs forms.
- Data Loading Module: Data loading module that uses the forms specified above for the creation of records that are stored locally.

## Style Guidelines:

- Primary color: Yellow (#FFFF00) to evoke energy and cheerfulness.
- Background color: White (#FFFFFF), to create a clean and bright backdrop.
- Accent color: Orange (#FFA500) for calls to action and interactive elements to grab attention and signal interactivity.
- Font: 'Inter' sans-serif for body text, lending the app a modern, machined look; also using 'Inter' for headlines since it is suitable for both.
- A clean, structured layout with clear sections for user management, form creation, and data loading. Each module will have its own page, easily accessible from the main dashboard.
- Simple, recognizable icons to represent each module in the navigation. Icons should be related to their respective functions, such as users, forms, and data entries.
- Subtle transitions and animations to enhance user experience when navigating between modules or submitting data.
